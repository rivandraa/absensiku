# 📱 Aplikasi Absensi Digital

## 📌 Deskripsi

Aplikasi Absensi Digital adalah sistem pencatatan kehadiran pegawai berbasis mobile..
Aplikasi ini dirancang untuk mempermudah proses absensi secara cepat, akurat, dan real-time.

---

## 🚀 Fitur Utama

* ✅ Absensi masuk
* 📍 Validasi lokasi (GPS)
* 📅 Rekap kehadiran otomatis
* 📊 Laporan absensi
* 🔒 Sistem keamanan pengguna

---

## ⚙️ Cara Instalasi

1. Download file APK terbaru
2. Buka file APK
3. Izinkan instalasi dari sumber tidak dikenal (jika diminta)
4. Klik **Install**
5. Tunggu hingga selesai
6. Buka aplikasi dan login

---

## ⚠️ Persyaratan Sistem

* Android 6.0 (Marshmallow) atau lebih baru
* GPS aktif
* Koneksi internet

---

## ❗ Catatan Penting

* Aplikasi hanya dapat digunakan oleh pegawai resmi
* Pastikan GPS aktif saat melakukan absensi
* Gunakan aplikasi sesuai dengan ketentuan yang berlaku
